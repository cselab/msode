# msode

microswimmer ODE solver

## usage

in release mode:

	make mainTest
	./mainTest config0.cfg config1.cfg ...


in debug mode:

	debug=1 make mainTest
	./mainTest config0.cfg config1.cfg ...

where `config0.cfg` is the initial configuration of rigid 0 (see `config/test.cfg` for example).
